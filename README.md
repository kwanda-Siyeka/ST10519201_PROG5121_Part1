## Project information
This project is a Java based chat application (Part1) developed using Netbeans and Maven
the program consists of three main classes:
Login.java – This class handles user registration, validation, and login logic
MainApp.java – This class manages user input and program interaction
LoginTest.java – contains JUnit tests to verify the functionality of the Login class

---

## Features of the application include:
Username validation (must contain an underscore and be no more than 5 characters)
Password complexity validation (minimum 8 characters, including a capital letter, number, and special character)
South African phone number validation (must start with +27 and be no more than 12 characters)
User registration with feedback messages
Login functionality with success and error messages
Unit testing using JUnit to ensure all methods work correctly
This project demonstrates the use of Java methods, input validation, decision structures, and basic testing.
